# Money_Manager

Money Manager is a beginner friendly full stack finance tracking application built to learn modern web development using React, Node.js, Express and MongoDB. This project helps users manage income, track expenses, budgets, transaction history and visualize spending analytics while learning frontend-backend integration and database connectivity through a clean and responsive interface.

## Features

- User Authentication
- Add/Edit/Delete Transactions
- Income & Expense Tracking
- Budget Management
- Dashboard Analytics
- Responsive UI
- Category-wise Expense Tracking

- ## Tech Stack

### Frontend
- React.js
- React Router
- Axios
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Tools
- Git & GitHub
- Postman

- ## Installation

### Clone Repository

git clone https://github.com/AditiMittal-05/Money-Manager.git

### Frontend Setup

cd client
npm install
npm start

### Backend Setup

cd server
npm install
npm run dev
